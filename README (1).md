# Calcined clay geopolymer thesis — visualizations

Interactive visualizations for my MSc thesis on calcined clay-based geopolymers using waste clays from Şile, Istanbul.

## Live site

Once GitHub Pages is enabled, the visualizations will be live at:
`https://<your-username>.github.io/<repo-name>/`

## What's here

- **Three-stage experimental flowchart** — overview of the SAI screening (Stage 1), activator screening (Stage 2), and the final geopolymer mixtures (Stage 3).
- **Test & characterization map** — every test mapped to its sample group and current status.

More visualizations (SAI bar charts, sulfate attack curves, activator dosage results) will be added as data comes in.

## Materials

- **K1** — waste clay from foundry sand reclamation (Kumsan Döküm)
- **K4** — waste clay from paper/ceramic industry, high iron oxide
- **K5** — waste clay, lower iron oxide than K4

## Local viewing

Open `index.html` in any browser — no server needed.
